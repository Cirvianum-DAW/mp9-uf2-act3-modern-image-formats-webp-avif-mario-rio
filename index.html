Modern Image Formats

WebP

WebP is a modern image format that provides superior lossless and lossy compression for images on the web. Using WebP, webmasters and web developers can create smaller, richer images that make the web faster.

AVIF

AVIF is a new image format derived from the keyframes of AV1 video. It's royalty-free and widely supported by streaming services, like Netflix. AVIF offers significant file size reduction for images compared with JPEG or WebP.
